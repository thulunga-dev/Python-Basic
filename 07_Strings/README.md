# 07 - Strings

## 📖 What is a String?

A **string** is a sequence of characters enclosed in single quotes (`' '`), double quotes (`" "`), or triple quotes (`''' '''` or `""" """`).

Strings are used to store text and are **immutable**, meaning they cannot be modified after creation.

### Example

```python
name = "Python"

print(name)
```

Output

```
Python
```

## 🔹 Common String Methods

| Method | Purpose |
|---------|---------|
| `upper()` | Convert to uppercase |
| `lower()` | Convert to lowercase |
| `capitalize()` | Capitalize first letter |
| `title()` | Capitalize each word |
| `strip()` | Remove spaces |
| `replace()` | Replace text |
| `find()` | Find a substring |
| `count()` | Count occurrences |
| `split()` | Split into a list |
| `join()` | Join strings |
| `startswith()` | Check prefix |
| `endswith()` | Check suffix |
| `isalpha()` | Alphabetic check |
| `isdigit()` | Digit check |
| `isalnum()` | Alphanumeric check |

> 📌 See **`strings.py`** for complete examples of each method.

---

## 📝 Key Points

- Strings are immutable.
- Indexing starts at `0`.
- Negative indexing starts at `-1`.
- Use slicing to extract substrings.
- Most string methods return a new string.
